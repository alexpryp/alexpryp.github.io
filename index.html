Front-end web developer

Прыпышный Александр

Приветствую! Меня зовут Александр, я Front-end разработчик. С тех пор как я, будучи ещё школьником, впервые увидел в браузере "Hello world!" из набранного и открытого мною HTML-файлика, прошло уже много лет, но эта простая "магия" не оставила меня равнодушным и тяга к веб-разработке с годами только росла. В какой-то момент интерес перерос в хобби и в свободное время я стал учиться верстать и программировать. Более глубоко знакомиться с программированием и версткой я продолжил с изучения основ C/C++, HTML и CSS. В дальнейшем стал осваивать и другие технологии и инструменты.

На данный момент в работе я использую:

  • HTML5
  • CSS3, Flexbox, Grid Layout, SASS, Bootstrap
  • JavaScript (ES5, ES6)
  • Lodash, jQuery, React, JSX, JSON, Babel
  • Git, GitHub
  • Node.js, Gulp, Create React App, Autoprefixer, Uglify, Sourcemaps
  • Linux, Visual Studio Code, Sublime Text, Emmet
  • Photoshop, Avocode, PSDetch
